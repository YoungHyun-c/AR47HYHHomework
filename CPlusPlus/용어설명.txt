1. 키워드
	C++ 에서 키워드는 컴파일러가 지정한 예약이라고 합니다.
	궁극적으로 본다면 그런 예약어 전체를 키워드라고 부릅니다.
	우리는 키워드를 변수명 이런 것으로 사용할수가 없다.
	예약되어 있고 이걸 사용하면 에러가 납니다. (ex: int int;)

2. 자료형 == 타입
	정수 실수 문자 이런것들을 구분하기 위한 키워드로 자료형이라는 다시 하위분류가 존재하게 됩니다.
	int <= 정수

3. 타입기반 언어
	C++은 타입기반 언어이기 때문에 자료형이 없는 것을 허용하지 않고 자료형이 변화시킬 수도 없다.

3. 주소값 == 이름 == 위치
  실행 됐다면 모든건 램에서 내 프로그램이 실행된 프로세스의 영역안에 존재해야해.
  램이란 결국 n바이트의 1열이니까.
  모든 이름은 결국 3211 바이트 번째 식으로 해석되어야 한다.
